# TUBITAK-2209

TÜBİTAK 2209 projesi kapsamında yumurta kabuk kalitesi verisinin analizi, modelleme çalışmaları ve rapor çıktıları.

## İçerik
- Veri keşfi ve ön işleme notları
- Model karşılaştırmaları ve metrikler
- Özellik önemleri / açıklanabilirlik analizleri
- Çalışma sonuçları raporu
- Görseller

## Öne Çıkan Dosyalar
- `veriAnaliz/calisma_sonuclari.md`: Akademik dilde kapsamlı çalışma raporu
- `veriAnaliz/kabuk_kalitesi_calismasi.py`: Modelleme ve rapor üretim scripti
- `veriAnaliz/model_karsilastirma_*.csv`: Model kıyaslama tabloları
- `veriAnaliz/ozellik_onemi_*.csv`: Özellik önemleri
- `görseller/`: Analiz görselleri

## Notlar
- Ham veriler ve sentetik veri dosyaları depoya dahil edilmemiştir.
- `.venv` klasörleri depoda yer almaz.

## Kullanım
1. Gerekli bağımlılıkları kurun (Python, pandas, scikit-learn, vb.).
2. Veriyi `veriAnaliz/yumurta kalitesi kalite.xlsx` konumuna yerleştirin.
3. `veriAnaliz/kabuk_kalitesi_calismasi.py` dosyasını çalıştırın.
